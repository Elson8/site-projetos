    Algoritmo em Java

    O programa foi desenvolvido na linguagem Java, utilizando o NetBeans IDE 8.2 tanto para satisfazer os requisitos da produção e desenvolvimento deste trabalho, como na agilidade de aplicações futuras.

    Primeiramente o que o programa faz é ler a partir de um endereço IP, e do prefixo de Rede, que o usuário fornece. Ele verifica a qual classe pertence, classifica o IP de acordo com o primeiro octeto, e calcula a máscara de rede em formato decimal, e descobre a rede e broadcast caso pertença a classe "C", irá imprimir uma lista com todas as sub-redes para o bloco de notas feito.
