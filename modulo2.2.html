    Arquitetura e Organização de Computadores

    Identificar os requisitos de Hardware necessários para execução do software planejado no componente curricular Programação Orientada a Objetos e configuração do servidor web para geração de logs, em Laboratório de Redes de Computadores.

    Hardware utilizada para desenvolvimento

    Foi utilizado para o desenvolvimento do software um notebook Dell Inspiron 15 gaming modelo 7567-A20P com processador core i7-7700HQ Quad Core 7ª Geração, 6M Cache, 2.8 até 3.8 GHz com Max Turbo, memória Ram DDR3-SDRAM 8GB 2400 Mhz, HD de 1TB Sata de 5400 RPM, placa de vídeo NVIDIA® GeForce® GTX 1050Ti com memória de 4GB GDDR5, tela 15.6 IPS Full HD, sistema operacional Windows 10 Single 64 bits. Apesar das configurações apresentadas, o software desenvolvido utilizando o programa NetBeans IDE não utilizou muito recurso da máquina, pois é uma aplicação leve para desenvolvimento de códigos e compilação de programas que não requer demasiado poder de hardware. Abaixo segue especificações técnicas de hardware utilizado.


    Especificação Técnica:

  • Processador: i7-7700HQ Quad Core 7ª Geração, 6M Cache, 2.8 até 3.8 GHz com Max Turbo
  • Memória: DDR3-SDRAM 8GB 2400 Mhz
  • Disco Rígido: HDD 1 TeraBytes SATA
  • Placa de Vídeo: NVIDIA® GeForce® GTX 1050Ti 4GB GDDR5
  • Tela: 15.6 IPS Full HD
  • Placa de Rede: Gigabit Ethernet
  • OS: Windows 10 Single x64 bits

    • Hardware utilizada para execução

    Para teste da aplicação foi utilizado um notebook Dell 3560 com processador 3ª Geração Intel(R) Core(TM) i5-3230M CPU @ 2.60GHz com 3 M de cache, Memória Ram 6GB DDR3 1600MHZ BCC, HDD Sata 750 GB 7200 RPM, Adaptador gigabit Ethernet Realtek PCIe GBE Family Controller, adaptador de vídeo AMD Radeon HD 7670M e Intel(R) HD Graphics 4000, tela 15.6 Full HD com AntiGlar, adaptador Wireless Dell 1703 802.11 b/g/n, Bluetooth 4.0, Controladoras ATA/ATAPI IDE Intel(R) 7 Series Chipset Family SATA AHCI Controller, sistema operacional escolhido para testes foi Windows 10 Professional 64 bits por ser mais atual, possuir suporte da Microsoft e melhor utilização do hardware apresentado. O teste da aplicação não consumiu muito hardware, pois a mesma é bem leve.


    Especificação Técnica:

  • Processador: Intel(R) Core(TM) i5-3230M CPU @ 2.60GHz 3 M cache
  • Memória: 6GB DDR3 DIMM 1600MHZ BCC
  • Disco Rígido: HDD 750 GB SATA 7.200 RPM
  • Placa de Rede: Gigabit Ethernet
  • Adaptador de Vídeo: AMD Radeon HD 7670M / Intel(R) HD Graphics 4000
  • Tela: 15.6 Full HD AntiGlar
  • Wireless: Dell 1703 802.11 b/g/n
  • OS: Windows 10 Professional x64 bits

    • Hardware para servidor

    Foi utilizado como servidor WEB o modelo PowerEdge T130 da Dell com Processador Intel® Xeon® E3-1220 v6 de 3 GHz, cache de 8 MB, Memória Ram UDIMM DE 8 GB 2400 MT/S, HD 1TB 7.200 RPM SATA 6Gbps 3.5in, controladora de rede gigabit ethernet Broadcom® BCM5720, 2 USB 3.0 e 4 USB 2.0 na traseira e frontal 1 USB 2.0 e 1 USB 3.0, monitor Dell SE2416 23”. Sistema operacional utilizado foi CentOS versão 7.5 para execução do Apache, bind e ftp server. A máquina escolhida para servidor compõe harmonia de hardware necessário e custo de aquisição, sendo a ideal para rodar as aplicações web e manter serviços de DNS e FTP.


    Especificação Técnica:

  • Processador: Intel® Xeon® E3-1220 v6 de 3 GHz, cache de 8 MB
  • Memória: UDIMM DE 8 GB 2400 MT/S
  • Disco Rígido: 1TB 7.200 RPM SATA 6Gbps 3.5in
  • Placa de Rede: Broadcom® BCM5720 Gigabit Ethernet
  • OS: CentOS 7.5
  • Dispositivo de Saída: Monitor Dell SE2416 23”
  • Dispositivos de Entrada: kit mouse teclado sem fio MK220
